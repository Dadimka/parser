Интерфейс текстового редактора

Пункт «Файл»

В пункте «Файл» меню текстового редактора реализован следующий функционал (см. рисунок А.1):

  • Создать новый документ
  • Открыть документ
  • Сохранить документ
  • Сохранить текущий документ в определённое место
  • Выйти из программы

Пункт «Правка»

В пункте «Правка» меню текстового редактора находится функционал для правки введённого текста (см. рисунок А.2):

  • Отменить изменения
  • Повторить последнее изменение
  • Вырезать выделенный фрагмент
  • Копировать выделенный фрагмент
  • Вставить фрагмент
  • Удалить выделенный фрагмент
  • Выделить все содержимое документа

Пункт «Текст»

Пункт меню «Текст» содержит команды для вывода информации о курсовой работе и программе (см. рисунок А.3):

  • Постановка задачи
  • Грамматика
  • Классификация грамматики
  • Метод анализа
  • Диагностика и нейтрализация ошибок
  • Тестовый пример
  • Список литературы
  • Исходный код программы

Пункт «Пуск»

При нажатии на пункт «Пуск» происходит запуск парсера.

Пункт «Справка»

Пункт меню «Справка» содержит:

  • «Вызов справки» — справочное описание программы
  • «О программе» — версия и информация о разработчике

Панель инструментов

Панель инструментов содержит кнопки вызова часто используемых пунктов меню (см. рисунок А.5):

  • Пуск парсера
  • Создание документа
  • Открытие документа
  • Сохранение текущих изменений
  • Отмена изменений
  • Повтор последнего изменения
  • Копировать выделенный фрагмент
  • Вырезать выделенный фрагмент
  • Вставить текстовый фрагмент
  • Очистить всё
