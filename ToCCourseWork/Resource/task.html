• Лямбда-выражение – это способ объявления анонимной (безымянной) функции в языке Python. Такие выражения позволяют компактно описывать простые функции, часто используемые как аргументы в других функциях.
  • Для объявления лямбда-выражения используется ключевое слово lambda.
  • Формат записи: имя_функции = lambda параметры: выражение
  • Пример
  • calc = lambda a, b, c: a + (b * c);
  • В связи c разработанной контекстно-свободной грамматикой G[Z] синтаксический анализатор лямбда-выражения будет считать верными следующие записи:
  • calc = lambda a: a;
  • calc = lambda a, b: a + b +43;
  • calc = lambda a, b, c: a + b + (c * 2);
