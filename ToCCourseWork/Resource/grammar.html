Грамматика λ-выражения Python G[<START>] (нотация Хомского)

  1. <START> → first_symb<V>
  2. <V> → symb<V>
  3. <V> → '=' <L>
  4. <L> → 'lambda' <ARG>
  5. <A> → ':' <E> <END>
  6. <ARG> → first_symb <ARG_N>
  7. <ARG_N> → symb <ARG_N>
  8. <ARG_N> → ',' <ARG>
  9. <ARG_N> → ':' <E> <END>
  10. <E> → <T> <A>
  11. <A> → ε | '+' <T> <A> | '-' <T> <A>
  12. <T> → <O> <B>
  13. <B> → ε | '*' <O> <B> | '/' <O> <B>
  14. <O> → <I> | <N> | '(' <E> ')'
  15. <N> → digit | digit<N>
  16. <I> → first_symb <I_R>
  17. <I_R> → symb <I_R> | symb
  18. <END> → ';'

Термы и нетермы

  • Z = <START>
  • VT = { a … z, A … Z, '_', 0 … 9, first_symb, symb, digit, '=', ':', ';', ',', '+', '-', '*', '/', '(', ')' }
  • VN = { <V>, <L>, <ARG>, <ARG_N>, <E>, <T>, <O>, <N>, <A>, <B>, <I>, <I_R>, <END> }

Примечания по токенам

  • first_symb → a … z | A … Z
  • symb → a … z | A … Z | '_' | 0 … 9
  • digit → 0 … 9
