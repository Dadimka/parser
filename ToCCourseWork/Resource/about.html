Функции пользователя

Работа с файлами

У пользователя есть следующие функции для работы с файлами:

  • Создать новый документ
  • Открыть документ
  • Сохранить документ
  • Сохранить как новый файл
  • Выход — завершает программу

Работа с текстом

Для работы с текстом реализованы следующие функции:

  • Отменить предыдущее изменение
  • Повторить изменение
  • Вырезать выделенный текст в буфер обмена
  • Копировать выделенный текст в буфер обмена
  • Вставить текст из буфера обмена
  • Удалить выделенный текст
  • Выделить весь текст из области редактирования файла

Приложение имеет справочную систему, запускаемую командой «Вызов справки».

Кнопка «Пуск» запускает парсер, который анализирует введённый текст.
